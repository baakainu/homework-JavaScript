## 1. html 마크업

일단 html을 시멘틱하게 마크업 하는 꿈을 꾸었습니다.
해보면 해볼수록 웹접근성은 정말 어렵다는 생각이 듭니다.
처음엔 div로 묶어버릴까 했지만, section을 사용하였습니다.
ul과 li를 사용해서 목록을 만들고,
article을 사용하여 각 슬라이드 내부 카드 콘텐츠가 독립적인 내용임을 나타냈습니다.

클래스를 지정할때는 어디서 주워들었던 BEM 네이밍을 사용했습니다.
정확히 알지도 못하고 사용하다가 두배로 공부한 느낌입니다...
그래도 정리된 html을 보니 어떤걸 하고 싶은지 명확히 보여서 기분이 좋습니다.

버튼도 aria-label을 사용해서 어떤 버튼인지 알려주고,
&lt 와 &gt 를 사용하는 걸 잊지 않았습니다.

css파일과 js파일을 만들어서 연결해준 후 css로 넘어갑니다.
